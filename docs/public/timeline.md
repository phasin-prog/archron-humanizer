## Timeline Rules

- Events are drawn from Timeline Event objects linked to Thinkers, Books, Concepts
- Each event shows: date, title, type icon, brief description
- Click event: opens detail card (not navigation)
- Double-click event: navigates to the Object page
- Filters persist across sessions (localStorage)
- Zoom levels: 1y, 5y, 10y, 50y, 100y
- Empty periods show no events (no placeholder)
- Mobile: vertical list instead of horizontal axis
- Timeline supports keyboard navigation (Arrow keys to move along axis)
