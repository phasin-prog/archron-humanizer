## Guide Creation

Guides are created in Studio by Writers/Editors:

```
Guide
├── Title + Description
├── Difficulty (beginner / intermediate / advanced)
├── Estimated duration
├── Prerequisites (optional: other Guides or Concepts)
└── Lessons (ordered list)
    ├── Lesson 1: Title + Body (Markdown + Object references)
    ├── Lesson 2: Title + Body
    └── ...
```

Each lesson can include any Object type via wikilinks:

```
In lesson body:
[[concept:shadow]] renders as an interactive Concept Card
[[thinker:carl-jung]] renders as a Thinker mini-profile
[[book:psychological-types]] renders as a Book reference
```

## Guide Discovery

- Featured on homepage (3 guides)
- Listed in Explore under each Discipline
- "Continue Learning" section on user Dashboard
- Recommended based on completed guides
- Each guide has a dedicated page with full lesson listing

## Guide Rules

- Guides are sequential — lessons must be completed in order
- Progress is saved per user (logged-in)
- Each lesson takes 5-15 minutes to complete
- Guides can reference other guides as prerequisites
- Completed guides are marked on the user Profile
- No downloadable PDF — the Guide lives on the platform with live links
- Future: quizzes at the end of each guide
