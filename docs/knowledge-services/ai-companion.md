# AI Companion

Milestone 10.6 — Optional. Build after all Level 1-4 services are complete.

## Role: Reader Companion

Not a chatbot. A **knowledge assistant**.

## Capabilities

| Capability | Level Required | Description |
|------------|----------------|-------------|
| Explain Concept | 4 (RAG) | "Explain the Shadow in simple terms" — retrieve Concept + summarize |
| Summarize | 4 (RAG) | "Summarize this article" — retrieve + condense |
| Compare | 4 (RAG) | "Compare Jung and Freud" — retrieve both + compare |
| Suggest Next | 2 (Rule) | "What should I read after Shadow?" — graph traversal |
| Translate | 3 (LLM) | "Translate this to Thai" — LLM with context |
| Quiz Me | 2 (Rule) | Generate quiz from current Object metadata |
| Outline | 2 (Rule) | Extract headings from Article into outline |

## AI Provider Architecture

```
Companion
    │
    ▼
Abstraction Layer
    │
    ├── Provider: OpenAI (GPT-4)
    ├── Provider: Anthropic (Claude)
    ├── Provider: Google (Gemini)
    ├── Provider: DeepSeek
    └── Provider: Local (Ollama)
    │
    ▼
Knowledge Engine (always consulted first)
    │
    ▼
Response
    └── Always includes citations
```

## Response Format

Every AI response follows this structure:

```json
{
  "answer": "The Shadow is an unconscious archetype...",
  "citations": [
    {
      "type": "concept",
      "slug": "shadow",
      "title": "Shadow",
      "url": "https://archron.com/concept/shadow"
    },
    {
      "type": "book",
      "slug": "man-and-his-symbols",
      "title": "Man and His Symbols",
      "url": "https://archron.com/book/man-and-his-symbols"
    }
  ],
  "confidence": "high",
  "disclaimer": "This answer was generated by AI using verified ARCHRON sources."
}
```

## Privacy

- Companion conversations are stored locally (not on server)
- No conversation data is used for training
- Users can clear conversation history at any time
- Companion can be disabled entirely in settings
- Provider selection is visible to the user
- No data sent to AI provider if Local (Ollama) is selected

## Acceptance Criteria

- [ ] Companion appears on every page as a non-intrusive icon
- [ ] Every answer cites at least one ARCHRON Object
- [ ] Companion can explain, summarize, compare, and suggest
- [ ] Companion does not answer questions outside ARCHRON's knowledge
- [ ] Privacy: conversations are not stored on server
- [ ] Companion can be disabled
