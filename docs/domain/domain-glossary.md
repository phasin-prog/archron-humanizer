# Domain Glossary

Canonical definitions of all domain terms used across the ARCHRON platform.

## A

**Achievement** — An earned recognition for completing a significant contribution milestone. Displayed on the User's Profile.

**Administrator** — A system-level role with full access to users, roles, configuration, and moderation.

**Alias** — An alternate name or search term registered to an Object, enabling discovery via multiple wordings or languages.

**Archive** — A content state for previously-published Objects removed from search results while preserving their URL.

**Article** — A long-form analytical or interpretive essay rendered through the reading system.

## B

**Backlink** — A reference from one Object to another, tracked and displayed as reverse relationships on the target Object.

**Book** — A published reference work or primary source in the bibliography.

## C

**Collection** — A user-curated grouping of Objects for personal or shared reference.

**Comment** — A user-submitted note or discussion on any published Object.

**Concept** — A core psychological or philosophical idea catalogued in the system.

**Content State** — The lifecycle stage of an Object: Draft, Review, Published, or Archived.

## D

**Discipline** — A broad domain of knowledge that contains Schools and Thinkers (e.g., Psychology, Philosophy).

**Draft** — An in-progress Object visible only to its Writer and invited Reviewers.

## E

**Editor** — A trusted Writer with authority to publish, archive, and manage content.

**Entity** — A distinct object in the domain model with a unique identity and lifecycle.

## G

**Glossary** — A set of defined terms within a specific domain.

**Guide** — A structured learning journey composed of sequential lessons referencing other Objects.

## K

**Knowledge Engine** — The core business-logic layer that resolves Objects, relationships, permissions, and taxonomy queries.

**Knowledge Graph** — The typed relationship network connecting all Objects.

## L

**Level** — A progression tier in the reputation system, granting a title badge.

## M

**Media** — A binary asset (image, audio, video) stored in object storage and referenced by Objects.

**Member** — A registered user with a Profile, reading history, and ability to create Collections.

## N

**Notification** — A system-triggered message informing a User of an event (publish, review request, achievement).

## O

**Object** — The atomic unit of knowledge. Everything in the system is an Object.

## P

**Profile** — A User's public-facing page displaying their bio, contributions, achievements, and Collections.

**Published** — A reviewed and approved Object that is live, indexed, publicly accessible, and immutable.

## Q

**Quote** — A notable passage from a Thinker or Work, stored as a linkable Object.

## R

**Reference** — An academic source or citation that supports content accuracy.

**Renderer** — The Markdown-to-Component pipeline that parses source, resolves wikilinks, and maps AST nodes to registered React Components.

**Reputation** — A numeric score reflecting a User's meaningful contributions to the platform.

**Review** — A content state where a Draft has been submitted for human validation.

**Reviewer** — A domain-expert Writer who validates content accuracy, citations, and academic quality.

**Role** — A permission level defining what actions a User can perform.

## S

**School** — An intellectual tradition or school of thought that groups Thinkers and Concepts.

**Search Engine** — The full-text and alias-based discovery system.

**Slug** — A URL-safe, immutable identifier for an Object.

**Symbol** — A sign, image, or archetype with defined meaning within the knowledge system.

## T

**Taxonomy** — The hierarchical classification system: Discipline → School → Theory → Concept.

**Term** — A single defined word or phrase within a Glossary.

**Theory** — A structured framework or set of explanatory principles within a School.

**Thinker** — A philosopher, psychologist, or intellectual whose work is catalogued.

**Timeline Event** — A dated historical occurrence linked to Thinkers, Concepts, or Schools.

## U

**User** — A registered entity with authentication credentials, a Profile, and a Role.

## W

**Wikilink** — A Markdown link syntax (`[[type:slug]]`) that resolves to an Object via the Knowledge Engine.

**Writer** — A Member who creates and submits content for review.
