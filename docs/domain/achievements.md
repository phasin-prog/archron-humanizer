# Gamification

Meaningful contribution tracking — not pointless point accumulation.

## Contribution Actions

| Action | Reputation | Description |
|--------|------------|-------------|
| Write Article | +50 | First publish of an original Article |
| Write Concept | +30 | First publish of a new Concept |
| Write Thinker | +30 | First publish of a new Thinker profile |
| Add Reference | +10 | Add a valid academic Reference |
| Submit Review | +20 | Complete a content Review |
| Get Approved | +15 | Article passes Review |
| Edit Draft | +5 | Substantial edit to a Draft |
| Create Guide | +40 | Publish a complete Guide |
| Create Collection | +5 | Curate a Collection of Objects |
| Report Issue | +5 | Flag a content problem |
| Fix Broken Link | +10 | Repair a dead or incorrect link |

## Achievements

| Achievement | Requirement | Type |
|-------------|-------------|------|
| First Words | Publish first Article | Milestone |
| Knowledge Builder | Publish 10 Articles | Milestone |
| Domain Expert | Publish 5 Articles in one Discipline | Expertise |
| Reference Master | Add 50 References | Contribution |
| Reviewer | Complete 10 Reviews | Contribution |
| Guide Maker | Publish 3 Guides | Creativity |
| Networker | Link 100 Objects | System |
| Cartographer | Create 20 Collections | System |
| Mentor | Have 5 Writers credit you as influence | Social |
| Centurion | Reach Level 5 | Progression |
| Editor's Choice | Article selected by Editor as featured | Recognition |

## Gamification Rules

- Reputation is never lost or deducted
- Achievements are awarded once per account
- Achievements are displayed on the User's public Profile
- Level is displayed as a badge next to the username
- System actions (Achievements, Level-ups) trigger Notifications
- No leaderboards — gamification is personal progress, not competition
- Gamification must never incentivize low-quality contributions
- Editors and Administrators do not earn reputation for their editorial actions
