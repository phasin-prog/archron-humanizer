# Brand Identity

## Brand Feel

ARCHRON must feel:

| ✅ Yes | ❌ No |
|--------|-------|
| Academic | Corporate |
| Modern | Startup |
| Calm | Dashboard |
| Premium | AI |
| Timeless | Cyberpunk |
| Minimal | Fantasy |
| Human | Gamified |
| Intelligent | Noisy |

## Brand Personality

- **Voice**: Authoritative but accessible. Academic but not cold. Never condescending.
- **Tone**: Calm, clear, precise. No hype. No urgency. No pressure.
- **Audience**: Lifelong learners, students, researchers, curious minds.
- **Promise**: Reduce the time between Question and Understanding.

## Visual Identity

- **Typography-led**: Text is the primary visual element. Everything else supports it.
- **Generous whitespace**: Content breathes. Margins are wide. Nothing is crowded.
- **Subtle warmth**: Not cold academic grey. Warm paper tones.
- **Academic references**: Citations, margins, notes feel scholarly without being old-fashioned.
- **Knowledge Object icons**: Every Object type has a distinct, recognizable icon.
- **No AI aesthetic**: No glowing gradients, glassmorphism, or "futuristic" elements.

## Logo

- Wordmark: "ARCHRON" in a clean, serif or neutral sans-serif
- Symbol (optional): A minimal node/constellation mark
- No acronyms, no taglines, no abstract shapes
- Monochrome-friendly
- Favicon: single letter or minimal symbol

## Tone of Voice

| Do | Don't |
|----|-------|
| "The shadow represents..." | "Unlock the power of your shadow!" |
| "This concept was developed by Jung in 1921." | "Jung dropped this mind-blowing concept in 1921 🔥" |
| "Related concepts include Persona and Anima." | "Check out these related vibes." |
| Use precise language | Use marketing superlatives |
| Use "we" for the platform | Use "our" possessive language |
| Let the content speak | Add unnecessary commentary |
