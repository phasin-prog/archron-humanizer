# Academic Seals Design System

Version: 1.0  
Status: Active  
Purpose: Visual identity system for ARCHRON's achievement recognition

---

## ปรัชญาการออกแบบ

### ARCHRON ไม่ใช่เกม

Achievement system ของ ARCHRON ไม่ใช่ระบบเกมมิฟิเคชัน

ไม่ใช่ gaming badges

ไม่ใช่ระบบแข่งขัน

**ARCHRON เป็นห้องสมุดวิชาการดิจิทัล**

ดังนั้น Achievement system ต้องเป็น **Academic Seals (ตราประทับวิชาการ)**

---

## Academic Seals คืออะไร

Seals เป็นตราประทับที่ใช้ในโลกวิชาการมาหลายศตวรรษ

- University emblems — ตราสัญลักษณ์มหาวิทยาลัย
- Library stamps — ตราประทับห้องสมุด
- Manuscript certifications — การรับรองต้นฉบับ
- Museum marks — เครื่องหมายพิพิธภัณฑ์
- Academic honors — เกียรติยศทางวิชาการ

Seals ไม่ใช่การตกแต่ง

Seals คือการรับรองคุณค่า

Seals คือการยืนยันความสำเร็จทางความรู้

---

## การใช้สีตาม Design Constitution

### Color Progression System

ARCHRON ใช้ระบบสีที่สื่อถึงการเติบโต ไม่ใช่ระดับเกม

```
Level 1-2: Slate      #465264   — Seeker, Apprentice
Level 3-4: Blue       #5F8DCE   — Scholar, Sage
Level 5-6: Silver     #C7D0DB   — Luminary, Guardian
Level 7:   Gold       #B89A63   — Architect
```

### หลักการใช้สี

- **Slate** — เริ่มต้น การก่อตัว การสำรวจ
- **Blue** — การเรียนรู้อย่างจริงจัง การวิจัย ความลึก
- **Silver** — ความโดดเด่น ความชัดเจน ความเป็นผู้นำ
- **Gold** — ความเป็นเลิศสูงสุด การสนับสนุน Premium

ไม่ใช่ Bronze/Silver/Gold แบบเกมโอลิมปิก

เป็นการไล่ระดับสีที่สื่อถึงวิวัฒนาการของนักวิชาการ

## Visual Style Rules

### ✓ ใช้

- **Shape:** Circle (○), Octagon (⬡), Hexagon (⬢), Diamond (◇), Compass (✦)
- **Style:** Flat SVG, outline only
- **Stroke:** 2px uniform, no variation
- **Color:** Monochrome per level (Slate/Blue/Silver/Gold)
- **Background:** Transparent
- **Typography:** Inter, uppercase, 10-12px

### ✗ ห้ามใช้

- ❌ Gradient fills
- ❌ Drop shadows
- ❌ 3D effects
- ❌ Bright colors
- ❌ Gaming aesthetics (stars, explosions, sparkles)
- ❌ Cartoon style
- ❌ Decorative illustrations
- ❌ Multiple colors in one seal

## Seal Structure

### รูปแบบพื้นฐาน

```
      ARCHRON
 ✦────────────✦
      ⌘
   [Title]
──────────────
  [Subtitle]
```

### องค์ประกอบ

1. **Header** — "ARCHRON" (บนสุด, เล็ก, tracking wide)
2. **Border** — รูปทรงเรขาคณิต (Circle/Octagon/Hexagon)
3. **Symbol** — สัญลักษณ์กลาง (abstract, semantic)
4. **Title** — ชื่อ Seal (ตัวหนา, ขนาดกลาง)
5. **Divider** — เส้นแบ่งบาง
6. **Subtitle** — รายละเอียดเพิ่มเติม (ถ้ามี)

## Academic Seals Catalog

### Level 1-2: Slate Seals (เริ่มต้นเส้นทาง)

#### 1. The Seeker
- **Symbol:** ○ with single dot (·)
- **Requirement:** สร้างบัญชีและอ่านบทความแรก
- **Shape:** Circle
- **Meaning:** จุดเริ่มต้น การเข้าสู่ห้องสมุด

#### 2. The Reader
- **Symbol:** ⌇ (three lines — pages)
- **Requirement:** อ่าน 10 บทความครบถ้วน
- **Shape:** Circle
- **Meaning:** การอ่านอย่างต่อเนื่อง

#### 3. The Collector
- **Symbol:** ▢ with three dots inside
- **Requirement:** บันทึกบทความลง Collection ครั้งแรก
- **Shape:** Hexagon
- **Meaning:** การเริ่มจัดระเบียบความรู้

### Level 3-4: Blue Seals (นักวิชาการที่ลงลึก)

#### 4. The Scholar
- **Symbol:** ⌘ (knowledge mark)
- **Requirement:** อ่าน 100 บทความครบถ้วน
- **Shape:** Octagon
- **Meaning:** ความเป็นนักวิชาการอย่างแท้จริง

#### 5. The Analyst
- **Symbol:** ◇ with center cross
- **Requirement:** อ่าน Object ครบทั้ง 5 ประเภท (Article, Concept, Thinker, Guide, Timeline)
- **Shape:** Diamond
- **Meaning:** ความเข้าใจโครงสร้างความรู้

#### 6. The Explorer
- **Symbol:** ✦ (compass)
- **Requirement:** อ่านบทความจาก 5 Domains ขึ้นไปI
- **Shape:** Circle (compass rose inside)
- **Meaning:** การสำรวจสาขาวิชาหลากหลาย

#### 7. The Archivist
- **Symbol:** ▭ with horizontal lines (archive)
- **Requirement:** สร้าง Collection 5 ชุดขึ้นไป
- **Shape:** Hexagon
- **Meaning:** ผู้จัดเก็บและจัดระเบียบความรู้

#### 8. The Cartographer
- **Symbol:** ⬢ with nodes (graph)
- **Requirement:** สร้าง Collection ที่มี Object มากกว่า 20 รายการ
- **Shape:** Hexagon
- **Meaning:** ผู้สร้างแผนที่ความรู้

### Level 5-6: Silver Seals (ผู้นำทางความรู้)

#### 9. The Curator
- **Symbol:** ⬡ with inner symmetry
- **Requirement:** Collection ของคุณถูก Featured โดย Editor
- **Shape:** Octagon
- **Meaning:** ผู้ดูแลคุณภาพความรู้ระดับสูง

#### 10. The Sage
- **Symbol:** ◇ with center dot radiating lines
- **Requirement:** อ่านบทความ 500+ และสร้าง Collection 10+
- **Shape:** Diamond
- **Meaning:** ผู้รอบรู้ที่ปฏิบัติอย่างต่อเนื่อง

#### 11. The Navigator
- **Symbol:** ✦ with orbital paths
- **Requirement:** ใช้ Knowledge Graph สำรวจ Concept มากกว่า 50 รายการ
- **Shape:** Circle
- **Meaning:** ผู้นำทางในโลกแห่งความคิด

#### 12. The Luminary
- **Symbol:** ○ with radiating lines (light source)
- **Requirement:** ถึง Level 5 และมี Collection Featured 3+
- **Shape:** Circle
- **Meaning:** ผู้ส่องสว่างให้ผู้อื่น

### Level 7: Gold Seals (เกียรติยศสูงสุดและการสนับสนุน)

#### 13. The Architect
- **Symbol:** ⬡ with geometric inner structure
- **Requirement:** ถึง Level 7 — Architect
- **Shape:** Octagon
- **Meaning:** ผู้สร้างสรรค์และออกแบบโครงสร้างความรู้

#### 14. The Companion
- **Symbol:** ✦ with inner circle (unity)
- **Requirement:** สนับสนุน ARCHRON ผ่าน Companion membership
- **Shape:** Compass star
- **Meaning:** ผู้สนับสนุน ผู้ร่วมเดินทาง

#### 15. The Patron
- **Symbol:** ◇ with crown element
- **Requirement:** สนับสนุน ARCHRON มากกว่า 1 ปี
- **Shape:** Diamond

## Animation Specifications

### หลักการ Animation

Seals ไม่ใช่เกม ดังนั้น animation ต้อง:

- ✓ เรียบง่าย สงบ มีคุณค่า
- ✓ เน้นความหมาย ไม่ใช่ความสนุก
- ✗ ไม่มี bounce, shake, หรือ explosion
- ✗ ไม่มี particle effects หรือ confetti

### Seal Unlock Animation

**Duration:** 300ms total

```
Phase 1: Fade In (100ms)
  opacity: 0 → 1
  
Phase 2: Outline Draw (150ms)
  stroke-dashoffset: animate
  
Phase 3: Scale Pulse (50ms)
  scale: 1.00 → 1.03 → 1.00
```

**Easing:** `cubic-bezier(0.4, 0, 0.2, 1)` — calm, natural

### Hover Animation

```css
transition: transform 150ms ease-out;
transform: translateY(-2px);
```

### Display Animation (Profile Gallery)

Seals ปรากฏแบบ stagger:

```
Seal 1: delay 0ms
Seal 2: delay 50ms
Seal 3: delay 100ms
Seal 4: delay 150ms
Seal 5: delay 200ms
```

## Implementation Guidelines

### SVG Structure

```svg
<svg width="64" height="64" viewBox="0 0 64 64" xmlns="http://www.w3.org/2000/svg">
  <g stroke="#5F8DCE" stroke-width="2" fill="none">
    <!-- Outer shape -->
    <circle cx="32" cy="32" r="28" />
    
    <!-- Inner symbol -->
    <path d="M32 20 L32 44 M20 32 L44 32" />
    
    <!-- Text (converted to path) -->
    <text x="32" y="8" font-size="6" text-anchor="middle">ARCHRON</text>
  </g>
</svg>
```

### Rendering Priority

1. **Profile overview:** แสดง 5 seals ล่าสุด
2. **Gallery view:** แสดงทั้งหมด เรียงตาม earned date
3. **Filter:** All / Earned / Locked
4. **Sort:** Recent / Level / Alphabetical

### Performance

- SVG ควร optimize ด้วย SVGO
- ใช้ `<symbol>` + `<use>` สำหรับ repeated seals
- Lazy load gallery view
- ไม่ต้อง preload seal images

## Technical Integration

### Data Structure

```typescript
interface AcademicSeal {
  id: string;
  slug: string; // 'the-seeker'
  name: string; // 'The Seeker'
  description: string;
  symbol: string; // SVG path or Unicode
  shape: 'circle' | 'octagon' | 'hexagon' | 'diamond' | 'compass';
  color: string; // hex color from constitution
  level: 1 | 2 | 3 | 4 | 5 | 6 | 7 | null; // null for special seals
  category: 'progression' | 'domain' | 'time' | 'support';
  requirement: string;
  earnedAt?: Date;
  isLocked: boolean;
}
```

### Database

```prisma
model AcademicSeal {
  id          String   @id @default(cuid())
  userId      String
  sealId      String   // reference to seal definition
  earnedAt    DateTime @default(now())
  
  user        User     @relation(fields: [userId], references: [id])
  
  @@unique([userId, sealId])
  @@index([userId])
}
```

### Award Trigger

```typescript
// Example: Award "The Scholar" seal
async function awardSeal(userId: string, sealId: string) {
  // Check if already earned
  const existing = await db.academicSeal.findUnique({
    where: { userId_sealId: { userId, sealId } }
  });
  
  if (existing) return;
  
  // Award seal
  await db.academicSeal.create({
    data: { userId, sealId }
  });
  
  // Silent notification
  await createNotification({
    userId,
    type: 'SEAL_EARNED',
    title: 'Academic Seal Earned',
    message: `You've earned ${getSealName(sealId)}`,
    priority: 'low'
  });
}
```

## Accessibility

### Color Independence

Seals ต้องเข้าใจได้แม้ไม่มีสี:

- ✓ ใช้ shape ที่แตกต่างกัน
- ✓ ใช้ symbol ที่ชัดเจน
- ✓ มี text label เสมอ

### Screen Readers

```html
<div role="img" aria-label="The Scholar seal, earned on January 15, 2026">
  <svg>...</svg>
</div>
```

### Focus States

```css
.seal:focus {
  outline: 2px solid #5F8DCE;
  outline-offset: 4px;
}
```

### Reduced Motion

```css
@media (prefers-reduced-motion: reduce) {
  .seal {
    animation: none;
    transition: none;
  }
}
```
