# Taxonomy

Not random tags — a structured classification system.

## Hierarchy

```
Discipline
    └── School
            └── Theory
                    └── Concept
```

## Example

```
Psychology
    └── Analytical Psychology
            └── Psychological Types
                    └── Introversion
```

```
Philosophy
    └── Existentialism
            └── Authenticity
                    └── Bad Faith (Mauvaise foi)
```

```
Anthropology
    └── Structural Anthropology
            └── Myth Structures
                    └── Mytheme
```

## Discipline → School Mapping

| Discipline | Schools |
|------------|---------|
| Psychology | Analytical Psychology, Psychoanalysis, Behaviorism, Humanistic, Cognitive, Gestalt |
| Philosophy | Existentialism, Stoicism, Idealism, Empiricism, Phenomenology, Pragmatism |
| Anthropology | Structural Anthropology, Cultural Anthropology, Symbolic Anthropology |
| History | Intellectual History, History of Ideas, Cultural History |
| Language | Structural Linguistics, Semiotics, Psycholinguistics |
| Mythology | Comparative Mythology, Myth Criticism, Archetypal Mythology |
| Religion | Comparative Religion, Theology, Mysticism |
| Science | Neuroscience, Systems Theory, Complexity Science |
| Symbolism | Hermeticism, Alchemical Tradition, Sacred Geometry |
| Art | Art Theory, Aesthetics, Visual Culture |
| AI & Future | Transhumanism, AI Ethics, Futures Studies |
| Civilization | Political Philosophy, Sociology, Legal Theory |

## Rules

- A School belongs to exactly one Discipline
- A Theory belongs to exactly one School
- A Concept can belong to multiple Theories
- An object can exist outside the taxonomy (e.g., Book, Article) but must still be tagged to at least one Domain
