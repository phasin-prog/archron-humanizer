# Knowledge Objects

Everything in the system must be an Object.

## Object Types

| Object | Description | Example |
|--------|-------------|---------|
| Concept | A core psychological or philosophical idea | Unconscious, Shadow, Ego |
| Thinker | A philosopher, psychologist, or intellectual | Jung, Freud, Nietzsche |
| Theory | A structured explanatory framework | Psychological Types, Structuralism |
| School | An intellectual tradition or school of thought | Analytical Psychology, Existentialism |
| Book | A published reference or primary source | Psychological Types, Beyond Good and Evil |
| Article | A long-form analytical or interpretive essay | The Archetypes of the Collective Unconscious |
| Symbol | A sign, image, or archetype with defined meaning | Mandala, Ouroboros, Cross |
| Timeline Event | A dated historical occurrence | Publication of The Interpretation of Dreams |
| Quote | A notable passage from a thinker or work | "The shadow is a moral problem" |
| Collection | A user-curated grouping of objects | My Jung Reading List |
| Guide | A structured learning journey of sequential lessons | Introduction to Analytical Psychology |
| Glossary | A set of defined terms within a domain | Glossary of Psychoanalytic Terms |
| Term | A single defined word or phrase within a Glossary | Complex, Archetype, Synchronicity |

## Principles

- Every object has a unique Slug
- Every object belongs to at least one Domain
- Every object can be linked to any other object
- Objects are polymorphic — a Thinker can also be a Concept
- No orphan objects — everything connects to something
