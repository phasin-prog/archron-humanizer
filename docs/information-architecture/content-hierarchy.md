# Content Hierarchy

Every Object type must have a standard content structure.

## Concept

```
Title
Definition (one sentence)
Overview
History / Origins
Key Characteristics
Related Concepts
Thinkers associated
Books
Timeline Events
References
```

## Thinker

```
Name
Image
Biography
Key Ideas
Major Works (Books)
Influenced By
Influenced
Schools
Timeline Events
Quotes
References
```

## Theory

```
Title
Definition
Overview
Core Concepts
Development History
Criticisms
Related Theories
Thinkers
Books
References
```

## School

```
Title
Definition
Historical Context
Core Principles
Theories
Key Thinkers
Major Works
Related Schools
References
```

## Book

```
Title
Original Title (if translated)
Author
Publication Year
ISBN
Summary
Themes
Key Concepts
Related Books
Thinkers
References
```

## Article

```
Title
Author
Published Date
Body (Markdown)
Related Concepts
References
```

## Symbol

```
Title
Image
Meaning
Origin
Cultural Context
Related Concepts
Appears In (Books, Art)
References
```

## Timeline Event

```
Title
Date (with precision)
Description
Significance
Related Thinkers
Related Concepts
Related Objects
References
```

## Quote

```
Quote Text
Source (Book / Article)
Page
Context
Thinker
Related Concepts
```

## Guide

```
Title
Description
Difficulty
Estimated Duration
Lessons (ordered list of Objects)
Prerequisites
Tags
```

## Collection

```
Title
Description
Curator
Objects (unordered list)
Visibility (public / private)
Tags
```
