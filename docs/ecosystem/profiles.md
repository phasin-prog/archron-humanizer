## Profile Sections

| Section | Visibility | Content |
|---------|------------|---------|
| Avatar + Name | Public | Name, role badge, level |
| Bio | Public | Short bio, expertise areas |
| Contributions | Public | Count of published objects by type |
| Achievements | Public | Earned achievement badges |
| Reputation | Public | Reputation score + level bar |
| Collections | Public (opt-out) | User's public collections |
| Reading Streak | Public | Consecutive days of activity |
| Recent Activity | Public | Last 10 actions (no private data) |
| Contact | Public (opt-in) | Website, email (optional) |
| Stats | Public | Views, guides completed |
| Saved Drafts | Private | Visible only to the user |
| Private Notes | Private | Visible only to the user |

## Profile Rules

- Profile is public by default but can be set to private
- Private profile hides contributions, achievements, collections
- Username is unique and can be changed once every 30 days
- Bio is limited to 280 characters
- Expertise tags are selected from the taxonomy
- No follower counts, no friends, no social feed
- Profile URL: `/profile/[username]`
- Deleted accounts: profile shows "Former contributor" with preserved contributions
