# Support System — ☕ Coffee Contribution

## Philosophy

**All content is free.**
**Knowledge is open.**
**The system is transparent.**

Support is not "buying content." Support is helping the system survive.

---

## What a Supporter is NOT

- ❌ Not a paywall
- ❌ Not premium content
- ❌ Not exclusive knowledge
- ❌ Not a class divide in knowledge access

---

## What a Supporter IS

### 1) Coffee Contribution Layer

Support = buying the system a coffee.

- No effect on content
- No effect on access
- No effect on knowledge ranking
- No effect on feature availability

### 2) Identity Badge (Light)

```
Guest  →  Member  →  Supporter ☕
```

Supporter is a **support status**, not a privilege tier.

### 3) Exclusive Interaction (Only One)

The only exclusive benefit:

**Direct Conversation with the Creator**

- Ask about system philosophy
- Discuss design decisions
- Ask about ARCHRON's direction
- Exchange insights and feedback

Not support content — **support dialogue**.

---

## Boundary Rules

A Supporter CANNOT:

- Edit articles (unless they are a Writer)
- Access unpublished Drafts
- See hidden/premium content
- Influence knowledge ranking
- Have privileges over other Readers

---

## Support Flow

```
Reader
   │
   ▼
Enjoy Content
   │
   ▼
Appreciate Value
   │
   ▼
Support Coffee ☕
   │
   ▼
Unlock:
   - Badge
   - Identity Label
   - Direct Chat Access
```

---

## Direct Chat Scope

### ✅ Allowed

- System philosophy and direction
- Concept meaning and context
- Design decisions behind ARCHRON
- Structural feedback
- Philosophical / knowledge discussion

### ❌ Forbidden

- Request to bypass content rules
- Request for exclusive information
- Feature leaks or unreleased info
- Admin or moderator privileges

---

## Philosophy (Core)

```
Knowledge must remain free,
but infrastructure is not free to maintain.
```

Support is gratitude in action.
Not a transaction for access.
