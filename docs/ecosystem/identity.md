## Connection Points

| From | To | How |
|------|----|-----|
| Knowledge | Studio | "Edit this" button on every Object |
| Knowledge | Guides | "Start learning" on every Object |
| Knowledge | Profile | Author attribution on every Object |
| Knowledge | Collections | "Save" button on every Object |
| Studio | Publishing | Draft вҶ’ Review вҶ’ Publish flow |
| Studio | Knowledge | New objects created feed into Knowledge |
| Guides | Profile | Completed guides shown on Profile |
| Companion | Knowledge | Companion suggests Objects to explore |
| Collections | Profile | User's collections visible on Profile |
| Community | Knowledge | Discussion linked to specific Objects |
| Support | Profile | Supporter badge on Profile (opt-out) |

## Ecosystem Rules

- Every product can reach every other product in вүӨ 2 clicks
- No product requires authentication to access (read-only)
- Authentication adds personalization (Collections, Progress, Profile)
- Data is shared across products вҖ” no silos
- Design language is consistent across all products
- Navigation is consistent across all products
