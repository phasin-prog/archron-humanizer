# Contribution System

Contribution comes in many forms — not just writing articles.

## Contribution Types

| Type | Description | Reputation |
|------|-------------|------------|
| **Write Article** | Publish an original Article | +50 |
| **Write Concept** | Create a new Concept | +30 |
| **Write Thinker** | Create a new Thinker profile | +30 |
| **Write Book** | Create a Book entry | +20 |
| **Write Symbol** | Create a Symbol entry | +20 |
| **Edit Content** | Substantial improvement to existing content | +10 |
| **Add Reference** | Add a valid academic Reference | +10 |
| **Submit Review** | Complete a content Review | +20 |
| **Get Approved** | Draft passes Review | +15 |
| **Translate** | Translate an Object to another language | +25 |
| **Correction** | Fix an error in published content | +15 |
| **Create Guide** | Publish a complete Guide | +40 |
| **Create Collection** | Curate a Collection (≥ 5 objects) | +5 |
| **Report Issue** | Flag a content problem | +5 |
| **Fix Broken Link** | Repair a dead or incorrect link | +10 |
| **Suggest Reference** | Propose a Reference for an Object | +5 |
| **Contribute Glossary** | Add terms to a Glossary | +10 |

## Quality Weights

Reputation is not linear — it is weighted by quality:

| Factor | Multiplier |
|--------|------------|
| Base reputation | 1.0× |
| Accepted on first review | 1.2× |
| Accepted with minor changes | 1.0× |
| Accepted after major rework | 0.8× |
| Cited by other content | +1.0× per citation (capped at 3.0×) |
| Corrected by peer (post-publish) | -0.5× |
| Flagged for quality issues | -1.0× |

## Contribution Rules

- Every contribution is attributed to the contributor on the Object page
- Contributions are immutable once published (corrections are new contributions)
- Editing own content gives reduced reputation (+5 instead of +10)
- Editors do not earn reputation for editorial actions (approve, publish, archive)
- Automated contributions (AI-assisted) earn reduced reputation
- Reputation is never negative — but can stop increasing until quality improves
- Contribution history is visible on the user's Profile
- All contributions are timestamped and versioned
